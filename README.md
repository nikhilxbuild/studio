🚀 Live Website

🌐 https://eduslide.in

EduSlide AI is a modern web tool built for students that converts messy notes and PDFs into clean, optimized, print-ready study sheets.

It allows users to upload PDFs, customize layouts, apply filters, and download structured study material in seconds.

---

✨ Features

📄 Smart PDF Processing

- Upload PDF notes instantly
- Convert notes into clean printable sheets
- Optimized page rendering
- Fast processing pipeline

📐 Layout Customization

- N-Up layout support
- Adjustable rows & columns
- Portrait / Landscape orientation
- Margin control (Default / Minimal / None)

🎨 Color Optimization

- Invert colors for printing
- Grayscale conversion
- Black & White mode
- Background whitening optimization

🧹 Auto Clean-Up Tools

- Remove blank pages automatically
- Crop borders from scanned PDFs
- Clean noisy backgrounds

⚡ Performance Optimized

- Smooth UI animations
- Fast processing workflow
- Optimized rendering pipeline
- Mobile-first performance

📱 Modern Interface

- Fully responsive UI
- Mobile-first design
- Glassmorphism components
- Smooth navigation system
